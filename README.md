# Portfolio 6: Lists
**Course:** BES10a  
**Student:** Ethan Seth S. Gratuito

## Project Description
This program is an Anime Watchlist Manager that demonstrates the core concepts of Python collections by storing, mutating, and organizing strings within a list structure.

The script fulfills the Chapter 8 List requirements by utilizing:
* **List Initialization & Appending:** Creating a base list and utilizing an indefinite loop with `.append()` to dynamically add new user inputs to the collection.
* **List Mutability:** Using the `.sort()` method to permanently change the order of the list into an alphabetized format.
* **Built-in Functions:** Applying the `len()` function to calculate and output the total number of elements contained in the list.
* **Logical Operations:** Using the `in` operator to search the list and execute conditional logic if a specific string is found.
* **Definite Iteration:** Combining the list with a `for` loop to systematically extract and print each element in the finalized collection.
